LHDN is rolling out mandatory e-Invoicing for Malaysian businesses. This page tracks the data you'll need (Tax ID, business registration, MSIC code) so you're ready when your turnover threshold requires submission — it does not submit invoices to LHDN's MyInvois system.
⚠️ What this is not: full e-Invoice submission requires registering with LHDN's MyInvois portal, a digital certificate, and real-time API integration (UBL/JSON format, validation, QR codes, credit notes). That is a separate project once CatDay's revenue threshold requires it. This page only makes sure the underlying data (TIN, BRN, MSIC) is captured now, so that integration is a much smaller lift later.
